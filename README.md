# Archivos de sesiones de clases web fundamentos

Cada carpeta representa una practica o proyecto distinto
un ejemplo seria una version alternativa de la pagina de la FIM de la UAS `pagina-uas/`

## Requisitos previos

* Tener instalado `Node.js` para usar vite y npm
> [!IMPORTANT]
> Se puede instalar desde la página oficial [Node.js](https://nodejs.org/es) pero recomiendo instalarlo con la herramienta [fnm](https://github.com/Schniz/fnm) que se usa desde la terminal y permite instalar ciertas versiones
> Instalar de preferencia la opcion LTS

* Tener Vscode como editor de código de preferencia
* Disfrutar el camino 👾

## Ejecutar aplicacion web

1. Ir al directorio de la app 

```bash
# cd <carpeta> 
cd pagina-uas/
```

2. Instalar las dependencias de Vite.js

> [!TIP]
> Si desean instalar pnpm para instalaciones en cache más rápidas instalen pnpm con el siguiente [enlace](https://pnpm.io/es/installation)


```bash
npm install
# recomiendo usar pnpm
# pnpm install
```

3. Ejecutar servidor de desarrollo

```bash
npm run dev
# pnpm dev
```
